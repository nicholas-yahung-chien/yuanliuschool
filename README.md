# Yuanliu School Static Mirror

This repository contains a static mirror of `https://yuanliuschool.com` (main site only).

## Scope

- Included: `https://yuanliuschool.com` pages and required visual assets.
- Excluded: `https://project01.yuanliuschool.com` (links are preserved as external links).
- External forms/services are kept as external links (Google Forms, YouTube, LINE, etc.).

## Current Mirror Snapshot

- Crawled pages: 141
- Discovered pages: 144
- Downloaded assets: 2729
- Failed assets: 9
- Metadata report: `site/_meta/summary.json`

## Folder Structure

- `site/`: static site output ready for Vercel deployment
- `site/assets/`: mirrored assets
- `site/_meta/`: crawl reports (`summary.json`, `pages.csv`, `assets.csv`, `route_map.json`, `assets_failed.csv`)
- `tools/mirror_site.py`: mirror crawler + rewriter
- `tools/validate_site.py`: static integrity checks

## Rebuild Mirror

```bash
python -m pip install -r requirements.txt
python tools/mirror_site.py --output site --delay 0.06 --timeout 25 --max-pages 500
python tools/validate_site.py --site-dir site
```

## GitHub Actions

- CI workflow: `.github/workflows/ci.yml`
  - Runs static link/asset validation on PR and `main` pushes.

- Vercel deploy workflow: `.github/workflows/deploy-vercel.yml`
  - PR: preview deployment
  - `main`: production deployment

Required GitHub repository secrets:

- `VERCEL_TOKEN`
- `VERCEL_ORG_ID`
- `VERCEL_PROJECT_ID`

## Vercel

`site/vercel.json` is generated by the mirror script.

The config enables:

- `cleanUrls` (keeps original URL style)
- `trailingSlash: false`
- automatic `rewrites` for pages whose original URL paths are too long for local Windows filesystem paths
